gyrogamma

PURPOSE

This scripts get gyromagnetic ratio of various spins (in radians)

SYNOPSIS

function g = gyrogamma(nucleus)

DESCRIPTION 

 This scripts get gyromagnetic ratio of various spins (in radians)
 INPUT    
  nucleus 'h1';'h2';'he3';'c13';'n15';'f19';'na23';'p31';'xe129';...

          'h1' is default
